# Task 07 - Standing Wave Modes

## Problem Statement

A standing wave with four antinodes is produced on a string of length $L = 80 \, \mathrm{cm}$.

Determine the wavelength of this standing wave.

## Theory

For a string fixed at both ends, the allowed standing-wave modes satisfy

$$
L = n \frac{\lambda}{2}
$$

where $n$ is the number of half-wavelength segments along the string.

For a mode with $n$ antinodes, the same relation applies because each antinode corresponds to one half-wavelength segment.

## Step-by-Step Solution

### 1. Convert the length to meters

$$
L = 80 \, \mathrm{cm} = 0.80 \, \mathrm{m}
$$

### 2. Use the condition for four antinodes

Since there are four antinodes,

$$
n = 4
$$

Then

$$
L = 4 \frac{\lambda}{2}
$$

Simplify:

$$
L = 2\lambda
$$

Therefore,

$$
\lambda = \frac{L}{2}
$$

Substitute $L = 0.80 \, \mathrm{m}$:

$$
\lambda = \frac{0.80}{2} = 0.40 \, \mathrm{m}
$$

## Final Result

The wavelength is

$$
\lambda = 0.40 \, \mathrm{m}
$$

## Interpretation

A standing wave with four antinodes contains four half-wavelength segments on the string. That means the total string length equals two full wavelengths, so the wavelength is half of the string length.
